Submitted for the 2025 Signal Processing Cup student competition hosted by the IEEE Signal Processing Society. The UCSD team was given a training set of real and deepfake images to implement a model for deepfake detection. Three models were used to test for the most accurate detection: Wavelet-CLIP, Inceptionv3 + Xception, and modified Inceptionv3.
